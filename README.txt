Приложение разработано в качестве тестового задания для компании "Бэкспар".
Инструкция по запуску:
- активировать docker-compose.yml
API:
POST /api/socks/income - принимает новую партию носков и регестрирует её или объединяет с существующей.
POST /api/socks/outcome - Забирает партию носков со склада
GET /api/socks - Возвращает количество носков по заданным параметрам.
PUT /api/socks/{id} - Обновляет партию носков по id.
POST /api/socks/batch - Принимает файл .xlsx, парсит данные о носках и регестрирует их или объединяет с существующими.
